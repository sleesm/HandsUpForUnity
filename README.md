## Description 설명
```
부모님을 대신하여 어린 아이들에게, Google Vision AI API를 적용해 단말기에 있는 카메라로,
그림에 맞는 텍스트의 정오답과 텍스트에 맞는 그림(혹은 사물) 정오답을 확인할 수 있는 앱입니다.
```
